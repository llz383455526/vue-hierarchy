# 增强交互 — 分类过滤、快速跳转、折叠记忆

## 概述

为 Vue Hierarchy 树视图增加高效的交互操作，让开发者在面对大型 Vue 组件时能快速定位到关心的内容。

## 动机

当前树视图固定展示所有分类，对于包含 10+ 分类、50+ 成员的大型组件，信息量过大，用户需要频繁滚动。缺少：
1. 按类别过滤的能力 — 无法只看 methods 而隐藏其他
2. 快速搜索跳转 — 内置 Ctrl+F 功能有限，无法按成员名模糊搜索并跳转
3. 折叠状态管理 — 每次刷新后折叠状态丢失，用户需要反复展开/收缩

## 变更范围

### Capability 1: 分类过滤 (category-filter)

在树视图标题栏添加过滤按钮，允许用户选择只显示特定分类（如仅 methods、仅 data）。

- 标题栏添加 filter 图标按钮
- 点击弹出 QuickPick 多选列表，列出当前文件的所有分类
- 选中的分类显示，未选中的隐藏
- 再次点击可重置为"显示全部"
- 过滤状态在切换文件时重置

### Capability 2: 快速跳转 (quick-jump)

提供 QuickPick 搜索功能，输入成员名即可模糊匹配并跳转到定义。

- 标题栏添加 search 图标按钮或绑定键盘快捷键
- 弹出 QuickPick 列出所有成员节点（带分类前缀，如 `[methods] handleSubmit`）
- 支持模糊搜索
- 选中后跳转到定义行并在树中高亮

### Capability 3: 智能折叠 (smart-collapse)

记忆折叠状态并提供批量折叠/展开操作。

- 记住用户对每个分类的折叠/展开状态，刷新时恢复
- 标题栏添加下拉菜单：全部展开 / 全部折叠 / 只展开某分类
- 折叠记忆按文件路径存储，切换文件时恢复对应状态
- 使用 workspaceState 持久化

## 非目标

- 不做跨文件搜索（仅当前文件）
- 不做自定义排序（保持源码顺序）

## 影响的文件

- `package.json` — 注册新命令和菜单
- `src/extension.ts` — 注册命令处理器
- `src/VueTreeProvider.ts` — 实现过滤逻辑、折叠状态管理
- `src/types.ts` — 可能需要扩展（如过滤状态类型）
